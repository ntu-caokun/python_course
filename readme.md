python课程ipynb文件

必备技能（自行学习）:
1. [安装vscode](https://zhuanlan.zhihu.com/p/698865320)：常用的开发工具
2. [安装anaconda](https://blog.csdn.net/weixin_43828245/article/details/124768518)：常用的环境管理工具
3. [git教程](https://liaoxuefeng.com/books/git/introduction/index.html)：代码版本管理工具

使用该文件夹：
1. git clone 该repo 
2. 新建环境 conda env create -f environment.yaml
3. 激活环境 conda activate python_course
4. 打开文件 jupyter notebook 
5. 选定章节文件
